## Dev

Install

```bash
pnpm i
```

dev

```bash
pnpm run dev
```

## Profile

- 😏 **Vite** for dev/build
- 😏 **wagmi** for web3 hook
- 😏 **Unocss** for CSS
- 😏 **Vercel** for preview

## Features

- 🌲 File system router
- 🏎️ AutoImport API
- 🛡️ TypeScript Checker & Eslint
- ✨ Custom Icon import by `unocss`
- 🍂 Tree shaking module
- 🚠 built in actions `.yml`
